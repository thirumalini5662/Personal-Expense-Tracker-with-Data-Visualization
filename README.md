# 💸 Personal Expense Tracker

## 📌 Project Overview
This is a Python-based expense tracker that helps users track, analyze, and visualize their spending.

## 🚀 Features
- Upload CSV file
- Automatic expense categorization
- Monthly filter
- Summary (Spend, Income, Savings)
- Bar chart and Pie chart visualization

## 🛠️ Tools Used
- Python
- pandas
- streamlit
- matplotlib

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   python -m streamlit run app.py

## 📊 Output
Interactive dashboard showing expenses and insights.
